# CLI Calculator

A C++ command-line calculator that supports:
- Addition
- Subtraction
- Multiplication
- Division (with divide-by-zero handling)
- Modulus (integer conversion)

## Run Locally

```bash
g++ CalculatorCLI.cpp -o CalculatorCLI
./CalculatorCLI
```
