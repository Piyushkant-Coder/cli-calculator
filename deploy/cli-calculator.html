CLI Calculator

A command-line calculator built in C++ that supports addition, subtraction, multiplication, division, and modulus with strong input validation.

View Repository See Source Code

Operations

+, -, *, /, and %.

Input Safety

Rejects invalid values and keeps prompting for a valid number.

Error Handling

Protects against divide-by-zero and modulus-by-zero operations.
